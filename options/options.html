About Color Blindness Types:

  • Protanopia: Difficulty distinguishing red and green colors, with red appearing darker.
  • Deuteranopia: Difficulty distinguishing red and green colors, but with normal brightness perception.
  • Tritanopia: Difficulty distinguishing blue and yellow colors.
  • Achromatopsia: Complete color blindness, seeing only in shades of gray.

Keyboard Shortcuts

Use these keyboard shortcuts to quickly access accessibility features

Available Shortcuts:

  • Alt+A: Open Accessibility Helper popup
  • Alt+C: Toggle high contrast mode on/off
  • Alt+Period: Increase text size
  • Alt+Comma: Decrease text size
  • Alt+R: Read the current page aloud
  • Alt+S: Stop reading

Note: These shortcuts can be customized in Firefox at about:addons → Accessibility Helper → Preferences → Keyboard Shortcuts.
